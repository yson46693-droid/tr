# Session Debug Logs

هذا المجلد يحتوي على سجلات الأخطاء المتعلقة بالجلسات.

## الملفات

- `session_debug.log` - سجل مفصل لجميع معلومات الجلسة وأسباب انتهائها

## كيفية القراءة

يمكنك فتح ملف `session_debug.log` لرؤية:
- معلومات الجلسة في كل طلب
- أسباب فشل `isLoggedIn()`
- أسباب إرجاع 401 من APIs
- محاولات إصلاح الجلسة

## تنسيق السجل

كل سجل يحتوي على:
- timestamp: وقت الحدث
- session_id: معرف الجلسة
- user_id: معرف المستخدم
- message: رسالة توضيحية
- context: معلومات إضافية
