# دليل استيراد العملاء من Excel

## المتطلبات

للاستفادة من ميزة استيراد العملاء من ملف Excel، يجب تثبيت مكتبة PhpSpreadsheet.

## التثبيت

### الطريقة 1: استخدام Composer (موصى به)

1. تأكد من تثبيت Composer على الخادم
2. قم بتشغيل الأمر التالي في مجلد المشروع:

```bash
composer install
```

أو إذا كان Composer غير مثبت، يمكنك تثبيت PhpSpreadsheet مباشرة:

```bash
composer require phpoffice/phpspreadsheet
```

### الطريقة 2: التثبيت اليدوي

1. قم بتحميل PhpSpreadsheet من: https://github.com/PHPOffice/PhpSpreadsheet
2. قم بفك الضغط ووضع المجلد في `vendor/phpoffice/phpspreadsheet`
3. تأكد من وجود ملف `vendor/autoload.php`

## تنسيق ملف Excel

يجب أن يحتوي ملف Excel على الأعمدة التالية:

| اسم العميل | رقم الهاتف | العنوان | الرصيد | المنطقة |
|------------|------------|---------|--------|---------|
| (مطلوب)    | (اختياري)  | (اختياري)| (اختياري)| (اختياري)|

### ملاحظات مهمة:

1. **الصف الأول** يجب أن يحتوي على رؤوس الأعمدة
2. **اسم العميل** هو العمود الوحيد المطلوب
3. يمكن استخدام أسماء الأعمدة بالعربية أو الإنجليزية:
   - اسم العميل / Name / Customer Name
   - رقم الهاتف / Phone / Mobile
   - العنوان / Address
   - الرصيد / Balance / Debt
   - المنطقة / Region
4. سيتم تخطي العملاء المكررين تلقائياً (بناءً على الاسم ورقم الهاتف والعنوان)
5. الحد الأقصى لحجم الملف: 10 ميجابايت

## كيفية الاستخدام

1. انتقل إلى صفحة "العملاء" → "عملاء الشركة"
2. اضغط على زر "استيراد من Excel"
3. اختر ملف Excel من جهازك
4. اضغط على "استيراد"
5. انتظر حتى يكتمل الاستيراد
6. سيتم عرض عدد العملاء المستوردين والخطأ إن وجد

## استكشاف الأخطاء

### خطأ: "مكتبة PhpSpreadsheet غير مثبتة"

**الحل:** قم بتثبيت PhpSpreadsheet باستخدام Composer كما هو موضح أعلاه.

### خطأ: "لم يتم العثور على عمود اسم العميل"

**الحل:** تأكد من أن الصف الأول يحتوي على عمود "اسم العميل" أو "Name" أو "Customer Name".

### خطأ: "حجم الملف كبير جداً"

**الحل:** قم بتقسيم الملف إلى ملفات أصغر (أقل من 10 ميجابايت لكل ملف).

## الدعم

إذا واجهت أي مشاكل، يرجى التحقق من:
1. تثبيت PhpSpreadsheet بشكل صحيح
2. تنسيق ملف Excel
3. صلاحيات الملفات والمجلدات
